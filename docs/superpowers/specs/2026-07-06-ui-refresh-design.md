# UI刷新 設計ドキュメント（静かなフォーカス系）

- Status: 承認済み（2026-07-06 ユーザー承認。実装は「音声UXバッチ」完了後）
- 起点: ユーザー評価「ダサすぎる」。毎日使う道具として気持ちいい見た目・操作感にする
- 方向性: **静かなフォーカス系**（余白・カード・落ち着いた配色、Linear/Things 的な道具感）
- 実装方式: **自前デザインシステム**（依存追加ゼロ。CSSカスタムプロパティ＋共有コンポーネント6個）

## 1. デザイントークン

`app/client/src/styles/tokens.css`（`:root` と `@media (prefers-color-scheme: dark)` でトークン差し替え）

- **配色**: ウォームグレー基調の3層（`--bg` 背景 / `--surface` 面 / `--border` 罫線）＋テキスト2階調（`--text` / `--text-muted`）＋アクセント1色（落ち着いたインディゴ系 `--accent`）。情報色 `--ok` `--warn` `--danger` は彩度控えめ
- **タイポグラフィ**: システムフォントスタック。英文チャンクは大きめ（17px〜）＋行間1.7（音読対象）、日本語glossは一段小さく `--text-muted`。タイマー数字は `font-variant-numeric: tabular-nums`
- **スケール**: 余白 4/8/12/16/24/32/48、角丸 8/12/16、影2層（控えめ）、トランジション150ms ease-out
- **アクセシビリティ**: コントラストAA、`:focus-visible` リング、`prefers-reduced-motion` で動きを無効化

## 2. 共有コンポーネント（`app/client/src/ui/`）

| コンポーネント | 責務 |
| --- | --- |
| `Button` | variant: primary/secondary/ghost/danger、loading（スピナー内蔵）、disabled |
| `Card` | 面＋罫線＋パディング。任意のヘッダ |
| `Screen` | 最大幅680の画面シェル。共通ヘッダ（戻る・ブロック進捗ドット・タイマーチップ） |
| `TimerChip` | 等幅数字のカウントダウン表示。残小時に色が情報として変化（演出ではない） |
| `ChunkList` | 英文太字＋日本語gloss＋🔊ボタンスロット（音声UXバッチの再生ボタンが挿さる） |
| `Banner` | info/warn/error の通知帯。現行の crimson テキストを全て置換 |

## 3. 画面ごとの再構成

- **StartScreen**: クイックドリル4枚をカードグリッド（タイトル＋分数）。強化セッション（60/30）は控えめな2次導線。自由会話はghostボタン。カレンダーは角丸ドットグリッド＋ホバーで日付表示。アンカーはインライン編集行
- **SessionRunner**: ヘッダ固定（ブロック進捗ドット＋TimerChip）。ブロック切替はフェードのみ
- **FourThreeTwoScreen**: 準備フェーズは Card（ChunkList＋骨組み＋モデルトーク）。ラウンド中は中央に大タイマー＋大録音ボタン（録音中は静かなパルス）。AEフィードバックは quote→issue→better→why_ja の階層カード
- **FreeTalk/Roleplay**: チャットバブル（自分=右・アクセント淡色、AI=左・面色）。メインボタンは大きく1つのまま
- **Shadowing/Reflection**: カード化、再生コントロール整理

## 4. 不変条件

- 機能・HTTP API・セッションイベント記録は完全不変（見た目とDOM構造のみ変更）
- ゲーミフィケーション制約（情報的フィードバックのみ）継続
- 依存追加ゼロ・外部フォント/CDN不使用（オフライン完全動作）
- ゲート: `cd app/client && bun run build`（tsc＋vite）、サーバテスト不変

## 5. 実装順序

音声UXバッチ（チャンク🔊再生・モデルトーク先読み＋本文表示・SQLiteライブラリ）完了後に、トークン→共有コンポーネント→画面を1つずつ移行。
